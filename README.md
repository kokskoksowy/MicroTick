# MicroTick
MicroTick is an ultra-lightweight Minecraft server engine compatible with many Minecraft versions, focused on extreme performance and minimal memory usage. It is capable of running on hardware with only a few MB of RAM on a single thread, and is specifically made for ESP32-S3 N16R8 microcontroller.
